03

技術指標判讀

看整股K線，找零股進出場點

① 均線 MA — 判斷趨勢

📈 多頭排列：5MA > 20MA > 60MA，適合偏多操作
📉 空頭排列：反之，不適合短線做多
💡 重點看 20MA（月線），多頭時回測不破是買點

② KD 指標 — 找進場點

🟢 K < 20：超賣區，考慮買進
🔴 K > 80：超買區，考慮賣出
⚡ 黃金交叉（K上穿D）→ 買進訊號
❌ 死亡交叉（K下穿D）→ 賣出訊號

③ 成交量 — 確認真假

✅ 放量上漲：訊號可信，買盤積極
⚠️ 縮量上漲：小心，後繼無力
🚨 爆量長黑：危險，主力出貨訊號

④ 布林通道 — 相對高低

🟢 股價碰 下軌：相對低點，可考慮買進
🔴 股價碰 上軌：相對高點，考慮獲利了結
💡 布林帶收窄後突破，往往是大行情起點

實戰判斷流程（條件符合越多，勝率越高）

// 進場前逐一確認
Step 1  →  60日均線向上      // 確認大趨勢多頭
Step 2  →  KD 低檔黃金交叉   // 等待買進訊號
Step 3  →  成交量放大確認    // 排除假突破
Step 4  →  接近布林下軌      // 確認相對低點

// 四個條件全中 → 高勝率進場點

風險控管

沒有風控，選股再準都會虧錢
-5%
停損點
跌破買入價5%立刻出場，不凹單、不等回來
+8%
停利點
達目標分批出場，不貪最高點，落袋為安
20%
單筆最大部位
單次不超過總資金20%，分散風險
3–5
同時持股數
持股太多顧不來，精選3到5檔專注操作

10萬元資金控管範例

// 總資金：$100,000
單筆最大投入  =  $20,000   （20%）
停損 -5%       =  虧 $1,000   （佔總資金 1%）
停利 +8%       =  賺 $1,600

// 就算連錯 5 次 → 只虧總資金 5%，不傷元氣
// 贏 3 次輸 2 次 → 仍能正報酬

操作紀律清單

✅ 進場前先設好停損價，掛好停損單
✅ 獲利達目標就出場，不貪最高點
✅ 每筆交易記錄原因與結果（建立交易日誌）
✅ 重大事件前（Fed、財報週）降低部位
❌ 不因「感覺會回來」而死撐不停損
❌ 不追漲殺跌，不情緒化操作
❌ 不在重大消息前重押單一個股
❌ 不all-in，不單壓一檔

短期零股操作 SOP

五步驟完整流程
1

篩選標的

用 Goodinfo / 財報狗 篩出零股成交量 >1,000股、股價 $100–$1,000、週振幅 >3%、非虧損的個股。

Goodinfo 財報狗 流動性 OK 基本面正常
2

技術面確認

大趨勢多頭（60MA向上）+ KD低檔黃金交叉 + 成交量放大 + 接近布林下軌，條件符合越多勝率越高。

均線多頭 KD黃金交叉 放量 近下軌
3

設定交易計劃

在進場前就設定好：進場價格、停損價（-5%）、停利目標（+8%）、部位大小（≤20%總資金）。

停損 -5% 停利 +8% 部位 ≤20%
4

執行紀律

13:40–14:30 盤後零股交易時間下單。嚴格照計劃執行，不受當下情緒影響，停損到就出，停利到就走。

13:40 下單 紀律執行 不凹單
5

記錄與檢討

每筆交易記錄：進場原因、技術訊號、結果（盈虧）。每週定期回顧，找出自己的盲點與改善點。

交易日誌 週回顧 持續優化

💡 核心心態： 單筆虧損是正常的，重點是整體勝率和賠率。目標是「勝率 50% + 平均獲利 > 平均虧損」， 長期下來就是正期望值。嚴格的風控不是膽小，而是讓你能持續留在市場的關鍵。
